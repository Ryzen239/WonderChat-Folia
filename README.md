Requirements

- A server running Folia, Paper, or Spigot
- Java 21 for Folia/Paper 1.21.x (Java 8 supported for older Paper builds)
- A MySQL database
- Vault
- PlaceholderAPI
- A permissions plugin (e.g. LuckPerms)
